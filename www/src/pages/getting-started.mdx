## Getting started

Tenon-UI can be installed in your project from the NPM repository.

It requires at least `React v16.3`.

### Installing with npm

```
npm install @tenon-io/tenon-ui
```

### Installing with yarn

```
yarn add @tenon-io/tenon-ui
```
